# Bild-Deck

Lege hier dein Karten-Deck als Bilddateien ab (z. B. `.gif`/`.png`/`.svg`,
benannt mit der Endung `.gif`). Dann in den Einstellungen **Kartendesign →
Bilder** wählen. Fehlt eine Datei, zeigt die App automatisch die gezeichnete
Karte.

> Wichtig: Lege nur Karten ab, an denen du die Nutzungsrechte hast.

## Namensschema

`<farbe>-<rang>.gif`

- **Farben:** `herz`, `ecken`, `schaufle`, `kreuz`
- **Ränge:** `6`, `7`, `8`, `9`, `10`, `bauer`, `dame`, `koenig`, `ass`
